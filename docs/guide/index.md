## About Squad

Squad enables teams to:
- **Define intelligent agents** with built-in GitHub Copilot integration
- **Coordinate multi-agent workflows** for complex tasks
- **Inherit and compose configurations** across teams and repositories
- **Monitor and observe** agent behavior with built-in telemetry

All with a simple, declarative configuration model and powerful Node.js SDK.
